ESAR

AR Education App

Year
2019
Time
2 Days
Type
Internship Project
Role
User Interface Design

This project was given to me as a part of my internship program. From which the designs then later, was given to final year students of Computer Application for their android application project.



The Goal

The goal was to design interface in such a way that it meets project requirements. This is an educational app which uses augmented reality as technology. This helps students in learning chemistry experiments in a better way. Here the user will scan the figure of the experiment from the book and detailed information about that experiment will be generated. The user will also be able to save the experiment for later reference. The home screen opens with the camera screen. Apart from that the user can also rate the app and can access usual user settings like change password.

05UI Design

User Interace Design

UI Design is the final product of the whole process. After a few sketch iterations and applying design language, the resulatnt is the interface design. This includes all the screens from begining to the finish line.
